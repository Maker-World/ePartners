## Features (Specifications)

| Features (Specifications) |                                |
| ------------------------- | ------------------------------ |
| Controller                | TM1638.                        |
| Display Type              | 8-Digit 7-Segment.             |
| LED                       | 8 Red Color 3mm.               |
| Key                       | 8 Tactile Switches.            |
| Supply Voltage            | 5V                             |
| Interface                 | 3 Lines Serial (CLK, STB, DIO) |
